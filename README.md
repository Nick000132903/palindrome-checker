# Palindrome Checker 🔄

A simple and elegant palindrome checker built with pure HTML, CSS, and JavaScript.

## 📋 About

This web application checks whether a word or phrase is a palindrome - meaning it can be read the same way forwards and backwards.

## ✨ Features

- Instant palindrome verification
- Clean and modern interface
- Automatic removal of special characters and spaces
- Empty input validation
- Responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Google Fonts

## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Type a word or phrase and click "Verify"

### 📝 Examples

- "racecar"
- "madam"
- "A man a plan a canal panama"
- "12321"

## 📱 Responsiveness

Fully responsive for desktop, tablet, and mobile devices.
